# Polish Travel Phrasebook 🇵🇱

A practical set of words and phrases for traveling in Poland. Each phrase has a rough English-style pronunciation in *[brackets]*. Stress almost always falls on the **second-to-last syllable**.

> **Good news:** In cities, tourist areas, hotels, and among younger people, English is widely understood. But even a few Polish words go a long way and locals genuinely appreciate the effort.

---

## Pronunciation Quick Guide

Polish looks scary but is consistent — once you learn the rules, words are read exactly as written.

| Letter(s) | Sounds like | Example |
|-----------|-------------|---------|
| `w` | **v** | *woda* = "voda" (water) |
| `ł` | **w** | *mało* = "mawo" (little) |
| `j` | **y** | *jutro* = "yutro" (tomorrow) |
| `c` | **ts** | *cena* = "tsena" (price) |
| `ch` / `h` | **h** (like loch) | *chleb* = "hlep" (bread) |
| `cz` | **ch** (church) | *czas* = "chas" (time) |
| `sz` | **sh** | *szybko* = "shipko" (fast) |
| `rz` / `ż` | **zh** (measure) | *rzeka* = "zheka" (river) |
| `ś` / `si` | soft **sh** | *coś* = "tsosh" (something) |
| `ć` / `ci` | soft **ch** | *być* = "bich" (to be) |
| `ą` | **on/om** (nasal) | *są* = "sont" (they are) |
| `ę` | **en/em** (nasal) | *się* = "sheng" |
| `ó` / `u` | **oo** | *góra* = "goora" (mountain) |
| `y` | short **i** (bit) | *ty* = "ti" (you) |

## Being Polite & Getting By

| English | Polish | Pronunciation |
|---------|--------|---------------|
| Do you speak English? | Czy mówisz po angielsku? | *[chi MOO-vish po an-GYEL-skoo]* |
| I don't speak Polish | Nie mówię po polsku | *[nyeh MOO-vyeh po POL-skoo]* |
| I don't understand | Nie rozumiem | *[nyeh ro-ZOO-myem]* |
| Can you help me? | Czy może mi pan/pani pomóc? | *[chi MO-zheh mee pan/PA-nee PO-moots]* |
| Please repeat | Proszę powtórzyć | *[PRO-sheh pov-TOO-zhich]* |
| Slower, please | Wolniej, proszę | *[VOL-nyey PRO-sheh]* |
| My name is... | Mam na imię... | *[mam na EE-myeh]* |
| Nice to meet you | Miło mi | *[MEE-wo mee]* |
| How are you? | Jak się masz? | *[yak sheh mash]* |
| I'm fine, thanks | Dobrze, dziękuję | *[DOB-zheh jen-KOO-yeh]* |

> **Tip:** Use *pan* (sir) / *pani* (madam) to be polite with strangers — Poles are quite formal.

## Money, Shopping & Numbers

The currency is the **złoty** (zł / PLN), divided into 100 *groszy*. Cards are accepted almost everywhere, even for tiny amounts.

| English | Polish | Pronunciation |
|---------|--------|---------------|
| How much is it? | Ile to kosztuje? | *[EE-leh to kosh-TOO-yeh]* |
| Too expensive | Za drogo | *[za DRO-go]* |
| Can I pay by card? | Czy mogę zapłacić kartą? | *[chi MO-geh za-PWA-cheech KAR-tom]* |
| Cash | Gotówka | *[go-TOOV-ka]* |
| Receipt | Paragon | *[pa-RA-gon]* |
| I'm just looking | Tylko oglądam | *[TIL-ko o-GLON-dam]* |
| Do you have...? | Czy macie...? | *[chi MA-cheh]* |

### Dishes worth knowing
- **Pierogi** — filled dumplings (the classic)
- **Żurek** *[ZHOO-rek]* — sour rye soup
- **Bigos** — hunter's stew (cabbage + meat)
- **Kotlet schabowy** — breaded pork cutlet
- **Gołąbki** *[go-WOMP-kee]* — stuffed cabbage rolls
- **Placki ziemniaczane** — potato pancakes
- **Zapiekanka** — open-faced baguette (street food)
- **Pączki** *[POONCH-kee]* — filled doughnuts
