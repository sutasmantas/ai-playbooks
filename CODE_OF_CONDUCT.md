# Code of Conduct

## Our pledge

We are committed to making participation in this project a respectful experience for everyone, regardless of background, experience level, or identity.

## Our standards

**Expected:**
- Giving and receiving feedback constructively
- Being direct about technical disagreements while staying respectful
- Citing evidence when making claims about what works
- Acknowledging when you're wrong

**Not acceptable:**
- Personal attacks or harassment
- Dismissing contributions without explanation
- Submitting fabricated research or made-up trial results

## Scope

This code of conduct applies to GitHub Issues, Discussions, Pull Requests, and any other project spaces.

## Enforcement

Report concerns to the project maintainer via GitHub. Reports are handled promptly and confidentially.
