# Research Angles: Evaluating LLM Instruction Following

**Topic:** How to evaluate whether LLMs follow instructions reliably — what benchmarks and methods exist.

---

## Angle 1: Instruction-Following Benchmark Surveys

**Vocabulary tradition / field:** NLP evaluation (ACL/EMNLP/NAACL benchmarking community)

**Search queries:**
- `"instruction following" LLM benchmark evaluation survey`
- `"instruction tuning" evaluation benchmark large language model`
- `LLM "follow instructions" metrics benchmark 2023 OR 2024`

**Best source types:** ACL Anthology papers, arXiv cs.CL preprints, NeurIPS/ICML proceedings

---

## Angle 2: RLHF and Preference-Based Alignment Evaluation

**Vocabulary tradition / field:** AI alignment / reinforcement learning from human feedback

**Search queries:**
- `RLHF "instruction following" evaluation reward model alignment`

- `InstructGPT alignment evaluation methodology`

**Best source types:** arXiv cs.AI/cs.LG preprints, OpenAI/Anthropic/DeepMind technical reports, NeurIPS papers
